Pitcher Scouting Report

This was developed to be used by batters in order to prepare for a specific pitcher they will face. The application allows batters to easily find a pitcher's past performance versus LHH and RHH so batters can better predict pitchers' actions at game time.

Enter a player's name in the search box to the left and click the search button in order to begin.
